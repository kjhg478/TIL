## React.FC
- React.FC를 이용해 얻을 수 있는 장점 2가지
    - children이라는 props가 기본적으로 탑재되어 있음 (children이 react.node로 되어 있어 별도로 값은 넣어주지 않아도 됨)
    - 몇 가지의 자동완성 기능

- React.FC의 단점
    - defaultProps가 제대로 작동하지 않음

- Readonly 속성
    - 읽기 전용 프로퍼티로 객체 생성 시 할당된 프로퍼티의 값을 바꿀 수 없다.
    - ex) readonly studentID : number;
